Memeinator3000

Status:

Closed Alpha

Memeinator3000 is an iOS app that allows you to easily change regular text into meme text. Memeinator3000 originally started as a small personal joke, but eventually grew into a marketable project. Memeinator3000 currently undergoing a Closed Alpha on Testflight. Memeinator3000 will be closed source until its 1.0 release, and then it will go open source.

Aer

Status:

Depreciated

Aer is a Discord bot written in Node, using the Discord.js library. Aer was originally designed to be a simple bot that would implement basic bot functions that other bot developers could use as a base to build their own bots on top of. Unfortunately, it grew to be a large and messy codebase, leading to the development of Aer 2.


Stack: UIKit Swift Firebase GitLab

Aer 2

Status:

Design Phase

Aer 2 is designed to fix the problems with and fulfill the original goal of Aer. Aer 2, just like the original, is built with Node and Discord.js. Aer 2 will have very little built in functionality, however will support plugins to add to its functionality. In order to help support developers, Aer 2 will include many first-party plugins to help simplify bot development.
